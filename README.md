## Quiosco Interactivo - NoSQL 

Proyecto de gestión de pedidos para un quiosco interactivo, desarrollado como parte de formación extra universitaria. Permite gestionar usuarios, productos, pedidos, valoraciones y proveedores, con consultas analíticas sobre ventas y rendimiento. Está dividido en tres secciones: el quiosco, la cocina y los proveedores.

## Tecnologías

- Bases de datos NoSQL: Google Colab-Python (MongoDB)
- Lenguaje: SQL queries

## Funciones

Quiosco interactivo
	⁃	Registro de usuarios
	⁃	Consulta de productos por categoría y catálogo completo
	⁃	Gestión de pedidos y detalle de compras
	⁃	Valoraciones y comentarios de productos
  - Consulta de alérgenos y coste energético de un producto
Cocina
	⁃	Consulta de productos por número de ventas 
	⁃	Gestión de trabajadores de cocina
  - Consulta de la composición de un producto
Proveedores
	⁃	Consulta de proveedores y productos que suministran 
	⁃	Consultas analíticas: ranking de productos, volumen de ventas por proveedor

## Rol
Proyecto en grupo: responsable del diseño del modelo de datos y las consultas para la gestión de cocina.
